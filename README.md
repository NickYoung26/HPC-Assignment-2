The aim of this assignment is to design and implement a simple class to perform operations with
vectors, using some of the object oriented features of Python 3. Then extend (inherit from) from
your class to make a new class with added functionality. Finally, make instantiated objects using
both classes to evaluate some simple problems in geometry.
